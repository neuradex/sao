# init-sao-claude

Quick setup for SAO (Stateful Agent Orchestration) with Claude Code.

## Usage

```bash
npx init-sao-claude
```

This will install:
- SAO core methodology (`.claude/sao/concepts/`)
- Claude Code commands (`.claude/commands/sao-starter.md`, `sao-step.md`)

## What is SAO?

SAO (Stateful Agent Orchestration) is a methodology for overcoming AI agent context limitations through document-based state persistence.

See [@neuradex/sao](https://www.npmjs.com/package/@neuradex/sao) for more details.
